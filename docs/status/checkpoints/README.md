# Human Checkpoint Records

This directory stores checkpoint briefs (prepared by agents) and results (recorded by humans).

## Naming Convention

- `hc-N-brief.md` — Agent-prepared review materials before checkpoint
- `hc-N-result.md` — Human decision after review

## Checkpoints

| HC-1 | Architecture Approval | 0 | ~45 min |
