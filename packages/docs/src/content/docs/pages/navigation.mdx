# Page Navigation

When a dashboard has multiple pages, users see navigation controls (tabs or a nav bar) that let them switch between pages. Navigation preserves filter state so users don't lose context when switching.

## Tips

- **Filter state persists** across page navigation — users don't need to reapply filters
- Use clear, descriptive page titles so users know what to expect on each page
- Configure [Click Actions](/interactions/click-actions/) to programmatically navigate users to specific pages
