# Click Actions

Click actions define what happens when a user clicks an element within a chart or widget. You can use click actions to filter other widgets, navigate to another page, or trigger custom behaviors.

<ExpandAll />

## Default Behavior

Without any click actions configured, clicking a chart element does nothing (or shows a tooltip).

## Configuration Options

### Filter Action

Clicking a chart element filters other widgets by the clicked value. This is [Cross-Filtering](/filters/cross-filtering/) functionality.

<FeatureScreenshot
  src={click_actions_panel_designer}
  alt="Filter click action"
  caption="Filter click action"
/>

### Navigate Action

Clicking a chart element navigates to another dashboard page, optionally passing context.

  src={interaction_editor_default_designer}

## Tips

- Use the [Interaction Editor](/interactions/drill-down/) to configure click actions visually
- Only configure click actions on charts where clicking a specific element is intuitive (bar charts, pie charts, table rows)
- Provide visual feedback so users know elements are clickable
