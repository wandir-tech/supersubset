# Alerts

The Alerts widget displays a list of operational alerts or notifications, color-coded by severity. Use it for monitoring dashboards, operations centers, or any scenario where users need to see a prioritized list of events.

## Default Appearance

An alerts widget with default settings shows a list of alert cards with severity badges and timestamps.

## Tips

- Place at the **top of a dashboard** for visual priority
- Use **maxItems** to keep the widget compact — 3–5 items is usually enough
- Use **wrap layout** for a horizontal banner style; **list layout** for a vertical sidebar style
- Pair with [Filters](/filters/select-filter/) to let users filter by severity
