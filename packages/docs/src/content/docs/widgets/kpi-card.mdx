# KPI Card

KPI cards display a single metric prominently, optionally with a comparison to a previous value showing trend direction. They're the most common widget for executive dashboards and summary rows.

A KPI card with default settings shows a large number with a title.

## Configuration Options

### Value Field (`valueField`)

The numeric field to display as the primary value.

<FeatureScreenshot
  src={kpi_card_props_designer}

### Comparison Field (`comparisonField`)

An optional field used to calculate and display the change from a previous period. Shows as an up/down arrow with percentage change.

<FeatureScreenshot
  src={kpi_card_props_designer}

### Format (`format`)

Controls number formatting: `compact` (1.2K, 3.4M), `currency` ($1,234.56), or default (1234).

## Tips

- Group **3â€“4 KPI cards** in a row at the top of your dashboard for an executive summary
- Use **comparison field** to show month-over-month or year-over-year trends
- Use **compact format** for large numbers to avoid visual clutter
- For target-based metrics, consider a [Gauge](/chart-types/gauge/) instead
