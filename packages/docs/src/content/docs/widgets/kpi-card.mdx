# KPI Card

KPI cards display a single metric prominently, optionally with a comparison to a previous value showing trend direction. They're the most common widget for executive dashboards and summary rows.

## Configuration Options

### Value Field (`valueField`)

The numeric field to display as the primary value.

<FeatureScreenshot
  src={kpi_card_props_designer}

  caption="Configure value field binding in the property panel"
/>

### Comparison Field (`comparisonField`)

An optional field used to calculate and display the change from a previous period. Shows as an up/down arrow with percentage change.

<FeatureScreenshot
  src={kpi_card_props_designer}
  alt="Comparison field showing trend"
  caption="Configure comparison field showing trend in the property panel"
/>

### Format (`format`)

Controls number formatting: `compact` (1.2K, 3.4M), `currency` ($1,234.56), or default (1234).

<FeatureScreenshot
  src={kpi_card_props_scrolled_designer}

  caption="Configure format options comparison in the property panel"

### Prefix (`prefix`)

A text prefix added before the value (e.g., `$`, `€`).

  src={kpi_card_props_scrolled_designer}
  alt="Currency prefix"

### Suffix (`suffix`)

A text suffix added after the value (e.g., `%`, `units`).

<FeatureScreenshot
  src={kpi_card_props_scrolled_designer}

### Font Size (`fontSize`)

Controls the size of the primary value display: `sm` (small), `md` (medium), or `lg` (large).

Controls how trend arrows are color-coded: `up-good` (green for increases) or `down-good` (green for decreases, useful for metrics like churn or bounce rate).

## Tips

- Group **3–4 KPI cards** in a row at the top of your dashboard for an executive summary
- Use **comparison field** to show month-over-month or year-over-year trends
- Use **compact format** for large numbers to avoid visual clutter
- For target-based metrics, consider a [Gauge](/chart-types/gauge/) instead
