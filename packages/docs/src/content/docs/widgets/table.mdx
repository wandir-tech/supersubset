# Table

Tables display data in a familiar rows-and-columns format with support for pagination, sorting, and column configuration. Use tables for detail views where users need to see individual records.

A table with default settings shows all available fields with automatic column widths and pagination.

## Configuration Options

### Page Size (`pageSize`)

The number of rows displayed per page. Smaller pages load faster; larger pages show more data at once.

## Tips

- Place tables at the **bottom of a dashboard** as a detail/drill-down view
- Use [Filters](/filters/select-filter/) to let users narrow down table data
- For a single summary number, use a [KPI Card](/widgets/kpi-card/) instead
