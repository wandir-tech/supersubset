# Markdown

The Markdown widget lets you add formatted text, instructions, descriptions, or documentation directly into your dashboard. Supports standard Markdown syntax including headings, lists, links, bold, italic, and code blocks.

## Tips

- Use Markdown widgets to **explain dashboard context** — add descriptions, methodology notes, or data freshness timestamps
- Place at the top of a dashboard page to set context before the charts
- Use headings (`##`) to create visual sections within the widget
- Links in Markdown can point to external resources or other dashboard pages
