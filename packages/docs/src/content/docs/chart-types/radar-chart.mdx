# Radar Chart

Radar charts (also called spider or web charts) display multiple dimensions radiating from a center point. Each axis represents a measure, and values are connected to form a polygon. Useful for comparing profiles or scores across multiple criteria.

## Configuration Options

### Dimensions (`dimensions`)

The list of measure fields, each mapped to a radar axis.

<ScreenshotComparison
  label="Dimension field bindings"
/>

### Series Field (`seriesField`)

The categorical field used to create multiple overlapping polygons for comparison.

<ScreenshotComparison
  label="Multiple series comparison"
/>

### Shape (`shape`)

Choose between `polygon` (straight edges) and `circle` (smooth) for the axis grid lines.

<ScreenshotComparison
  label="Shape: polygon vs circle"
/>

### Filled (`filled`)

When enabled, the area inside the polygon is filled with a semi-transparent color.

<ScreenshotComparison
  label="Filled radar areas"
/>

## Tips

- Limit to **5–8 dimensions** for readability
- Use **filled** mode when comparing two or three entities to see area overlap
- Ensure all dimensions use comparable scales — otherwise normalize your data
