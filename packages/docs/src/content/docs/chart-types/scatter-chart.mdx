# Scatter Chart

Scatter charts plot individual data points on a two-dimensional grid, ideal for exploring correlations between variables. Optionally encode additional dimensions through point size and color.

### Color Field (`colorField`)

An optional categorical field that colors points by group.

### Symbol Size (`symbolSize`)

Controls the base size of data points. Increase for sparse data; decrease for dense datasets.

## Tips

- Use **size field** to add a third dimension (e.g., revenue by cost and volume)
- Use **color field** to reveal clusters or groups in your data
- For dense datasets, reduce **symbol size** and enable transparency
- Combine with [Cross-Filtering](/filters/cross-filtering/) to click a point and filter related widgets
