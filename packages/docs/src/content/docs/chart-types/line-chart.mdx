# Line Chart

Line charts display data points connected by straight or curved lines, ideal for showing trends over time. In the Sales Dashboard, the Revenue Trend chart uses a line chart to show monthly revenue and cost.

## Configuration Options

### X-Axis Field (`xField`)

The field used for the horizontal axis — typically a time or category field.

<ScreenshotComparison
  label="X-axis field binding"
/>

### Y-Axis Fields (`yFields`)

One or more measure fields plotted as separate lines. Add multiple fields to compare trends.

<ScreenshotComparison
  label="Multiple Y-axis fields"
/>

### Smooth Lines (`smooth`)

When enabled, lines are drawn as smooth curves instead of straight segments. This produces a more polished look but can obscure precise data point positions.

<ScreenshotComparison
  label="Smooth lines enabled"
/>

### Area Fill (`area`)

When enabled, the area below each line is filled with a semi-transparent color. Useful for emphasizing volume or magnitude.

<ScreenshotComparison
  label="Area fill enabled"
/>

### Show Points (`showPoints`)

Toggle the visibility of data point markers on the line. Hiding points gives a cleaner look; showing them makes individual values easier to read.

<ScreenshotComparison
  label="Data points visibility"
/>

## Tips

- Use **smooth lines** for presentation dashboards where visual polish matters more than precision
- Enable **area fill** when you want to emphasize the magnitude of a trend
- Add a **second Y-axis field** to compare two related measures (e.g., revenue vs. cost)
- Combine with a [Date Filter](/filters/date-filter/) to let users zoom into specific time ranges
