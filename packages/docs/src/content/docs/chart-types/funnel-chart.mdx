# Funnel Chart

Funnel charts visualize sequential stages in a process, where each stage is proportionally narrower than the previous. Commonly used for conversion funnels (e.g., leads → opportunities → deals).

## Tips

- Use **descending sort** for classic conversion funnels
- Add **gap** spacing to emphasize individual stages
- Best with **3–7 stages** for clear visualization
