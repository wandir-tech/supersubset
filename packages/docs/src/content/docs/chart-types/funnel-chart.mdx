# Funnel Chart

Funnel charts visualize sequential stages in a process, where each stage is proportionally narrower than the previous. Commonly used for conversion funnels (e.g., leads → opportunities → deals).

## Configuration Options

### Value Field (`valueField`)

The numeric field determining each stage's width.

<ScreenshotComparison
  label="Value field binding"
/>

### Label Field (`labelField`)

The field used to label each funnel stage.

<ScreenshotComparison
  label="Label field binding"
/>

### Sort Order (`sort`)

Control whether stages are sorted by value (`descending`, `ascending`) or displayed in data order (`none`).

<ScreenshotComparison
  label="Sort order options"
/>

### Gap (`gap`)

Spacing between funnel segments. Increase for a cleaner separated look.

<ScreenshotComparison
  label="Gap between segments"
/>

## Tips

- Use **descending sort** for classic conversion funnels
- Add **gap** spacing to emphasize individual stages
- Best with **3–7 stages** for clear visualization
