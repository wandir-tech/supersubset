# Waterfall Chart

Waterfall charts show how an initial value is affected by a series of positive and negative changes, with a running total. They're often used for financial analysis — showing how revenue breaks down into profit through cost components.

<ExpandAll />

## Default Appearance

A waterfall chart with default settings shows floating bars indicating increases (green) and decreases (red) from a baseline.

### Category Field (`categoryField`)

The field labeling each step in the waterfall.

  label="Category field binding"

### Value Field (`valueField`)

The numeric field representing the change (positive or negative) at each step.

  label="Value field binding"

## Tips

- Ideal for **P&L analysis**, budget breakdowns, or step-by-step cost accounting
- Positive values extend upward; negative values extend downward
- The final bar typically shows the net total
