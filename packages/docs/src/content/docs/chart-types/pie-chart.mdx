# Pie Chart

Pie charts show how a whole is divided into parts, with each slice proportional to its value. Use pie charts when you have a small number of categories (ideally 2–7) and want to emphasize relative proportions.

### Donut Mode (`donut`)

When enabled, the center of the pie is hollow, creating a donut shape. This is often preferred for a cleaner, more modern look and can display a summary value in the center.

### Rose Type (`roseType`)

When enabled, slice radii vary by value — larger values have longer slices extending further from the center. This creates a "nightingale rose" effect that can make differences between values more visible.

/>

### Show Labels (`showLabels`)

Toggle the visibility of slice labels. Disable when using a legend or when slices are obvious from color coding.

## Tips

- Limit to **7 or fewer slices** for readability — group small values into "Other"
- Use **donut mode** for a modern aesthetic and to place a total or title in the center
- **Rose type** is useful when values are similar and you want to amplify differences
- For many categories, consider a [Bar Chart](/chart-types/bar-chart/) instead
