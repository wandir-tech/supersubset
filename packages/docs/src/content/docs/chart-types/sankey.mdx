# Sankey Diagram

Sankey diagrams visualize flows between nodes, where the width of each link is proportional to the flow quantity. Ideal for showing how values move between stages, categories, or systems.

<ExpandAll />

## Default Appearance

A Sankey diagram with default settings shows nodes as vertical bars connected by flowing links.

## Tips

- Works best for **multi-stage flows**: marketing channel → lead source → deal stage
- Keep the number of nodes manageable — group small flows into "Other"
- Data must not contain circular references (A → B → A)
