# Area Chart

Area charts are line charts with the region below the line filled in, emphasizing the magnitude of trends over time. They work especially well for showing volume, cumulative totals, or comparing the relative size of multiple series.

<ExpandAll />

## Default Appearance

An area chart with default settings shows filled regions under each line with semi-transparent colors.

## Configuration Options

### X-Axis Field (`xField`)

The time or category field for the horizontal axis.

### Y-Axis Fields (`yFields`)

One or more measure fields. Each field produces a filled area series.

<FeatureScreenshot
  src={area_chart_props_designer}

  caption="Configure multiple y-axis fields in the property panel"

### Smooth Curves (`smooth`)

When enabled, area boundaries are drawn as smooth curves.

<FeatureScreenshot
  src={area_chart_props_scrolled_designer}

  caption="Configure smooth area curves in the property panel"

### Opacity (`opacity`)

Controls the transparency of the fill. Lower values let overlapping areas show through; higher values produce more vivid fills.

<ScreenshotComparison
  label="Full opacity fill"

### Stacked Areas (`stacked`)

When enabled, areas are stacked so each series starts where the previous one ends. This shows cumulative totals and the contribution of each series.

## Tips

- Use **stacked areas** to show how components add up to a total over time
- Lower **opacity** when you have overlapping series to keep all areas visible
- For precise comparisons, a [Line Chart](/chart-types/line-chart/) may be clearer
