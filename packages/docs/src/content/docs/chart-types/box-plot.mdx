# Box Plot

Box plots (box-and-whisker diagrams) show the statistical distribution of values: median, quartiles, and outliers. Useful for comparing distributions across categories or groups.

<ExpandAll />

## Default Appearance

A box plot with default settings shows boxes spanning the interquartile range with whiskers extending to the min/max values.

## Tips

- Best for **comparing distributions** across 3–10 groups
- Look for differences in median, spread, and outliers between groups
- For single distributions, consider simple summary statistics in a [KPI Card](/widgets/kpi-card/)
