# Gauge

Gauges display a single numeric value against a defined range, similar to a speedometer. Ideal for KPIs that need to show progress toward a target or threshold.

<ExpandAll />

## Default Appearance

A gauge with default settings shows a half-circle dial with a needle pointing to the current value.

## Tips

- Set **min** and **max** to match your business context (e.g., 0–100 for percentages)
- Use multiple gauges side by side for a KPI dashboard row
- For simpler metric display, consider a [KPI Card](/widgets/kpi-card/) instead
