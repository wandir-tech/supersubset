# Heatmap

Heatmaps display data as colored cells in a grid, where color intensity represents the value at each intersection. Useful for spotting patterns, clusters, and outliers across two categorical or time dimensions.

## Tips

- Works best with **grid-like data** — daytime × weekday, category × region, etc.
- Choose a **sequential color range** for single-metric data (light to dark)
- Use a **diverging color range** when data has a meaningful midpoint (e.g., profit/loss)
- For sparse data, consider a [Scatter Chart](/chart-types/scatter-chart/) instead
