# Welcome to Supersubset

Supersubset is a visual dashboard builder that lets you create interactive analytics dashboards without writing code. You can drag and drop charts, tables, KPI cards, and other widgets onto a canvas, bind them to your data, configure filters and interactions, and publish a live dashboard for your users.

<ExpandAll />

## What You Can Build

With Supersubset, you can create dashboards that include:

- **14 chart types** — Line, Bar, Area, Pie, Scatter, Radar, Funnel, Gauge, Heatmap, Sankey, Treemap, Waterfall, Box Plot, and Combo charts
- **KPI cards** — Single-value metrics with trend comparisons
- **Data tables** — Sortable, paginated tabular views
- **Rich text** — Markdown-formatted content blocks
- **Alert widgets** — Operational alerts and notifications
- **Filters** — Global and scoped filters that update charts in real time
- **Cross-filtering** — Click a chart element to filter all related widgets
- **Multi-page dashboards** — Organize content across multiple pages with navigation
- **Import/Export** — Share dashboard definitions as JSON

## The Designer Interface

The Supersubset designer is a visual editor where you build your dashboards. It consists of:

1. **Widget Palette** (left) — Drag chart types, widgets, and layout components onto the canvas
2. **Canvas** (center) — The live preview of your dashboard layout
3. **Property Panel** (right) — Configure the selected widget's settings
4. **Toolbar** (top) — Undo/Redo, page management, code view, import/export
