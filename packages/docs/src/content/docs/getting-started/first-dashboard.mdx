# Your First Dashboard

This guide walks you through building a simple sales dashboard from scratch. By the end, you'll have a dashboard with KPI cards, a line chart, a bar chart, and a data table — all connected to filters.

<ExpandAll />

## Step 1: Open the Designer

When you open Supersubset, you'll see the designer interface with an empty canvas.

<ScreenshotComparison
  label="Empty designer canvas"
  designerSrc={overview_designer}
  designerAlt="Empty designer canvas in designer mode"
  viewerSrc={overview_viewer}
  viewerAlt="Empty designer canvas in viewer mode"
/>

## Step 2: Add KPI Cards

Drag three **KPI Card** widgets from the widget palette onto the canvas. Configure each one:

- **Total Revenue** — Value field: `revenue`, Format: `compact`, Prefix: `$`
- **Orders** — Value field: `orders`, Format: `compact`
- **Avg Order Value** — Value field: `aov`, Format: `currency`

<FeatureScreenshot
  src={first_dashboard_component_palette_designer}
  alt="KPI cards configured"
  caption="KPI cards configured"
/>

## Step 3: Add a Line Chart

Drag a **Line Chart** onto the canvas below the KPI cards. Configure it:

- X-axis: `month`
- Y-axis: `revenue`, `cost`
- Smooth lines: enabled

<FeatureScreenshot
  src={first_dashboard_layers_designer}
  alt="Line chart with revenue trend"
  caption="Line chart with revenue trend"
/>

## Step 4: Add a Bar Chart

Drag a **Bar Chart** next to the line chart. Configure it:

- X-axis: `region`
- Y-axis: `revenue`
- Horizontal: enabled

<FeatureScreenshot
  src={first_dashboard_toolbar_designer}
  alt="Horizontal bar chart by region"
  caption="Horizontal bar chart by region"
/>

## Step 5: Add a Data Table

Drag a **Table** widget below the charts. It will automatically display all available fields from your dataset.

<FeatureScreenshot
  src={first_dashboard_viewer_overview_viewer}
  alt="Data table with orders"
  caption="Data table with orders"
/>

## Step 6: Add Filters

Open **Dashboard Filters** from the header. Add:

- A **Select** filter for Region
- A **Select** filter for Category
- A **Date** filter for Order Date

These are dashboard-level filter definitions. The Filter Bar widgets on the page decide which authored filters viewers see at runtime.

<FeatureScreenshot
  src={first_dashboard_designer_overview_designer}
  alt="Dashboard filters configured"
  caption="Dashboard filters configured"
/>

## Step 7: Preview Your Dashboard

Switch to **Viewer** mode to see your finished dashboard. Try selecting different regions and categories to see the charts update in real time.
