# Grid

The grid is the foundational layout container for every dashboard page. It uses a 12-column system where widgets and rows are placed within the grid. All other layout components (rows, columns, widgets) live inside a grid.

## Configuration Options

### Columns (`columns`)

The number of columns in the grid (default: 12). Most dashboards use 12 columns for maximum flexibility.

## Tips

- Use the **12-column grid** to create layouts like: 4+4+4 (three equal), 8+4 (main + sidebar), 6+6 (half-half)
- Widgets specify their width in grid columns (e.g., `width: 6` fills half the grid)
- Rows within a grid group widgets on the same horizontal line
