# Header

Headers display title text to label sections of your dashboard. Use them to create visual hierarchy and help users navigate content.

## Tips

- Use a **large header** at the top of each page for the page title
- Use **medium/small headers** to separate sections within a page
- For rich text with formatting, use a [Markdown](/widgets/markdown/) widget instead
