# Divider

Dividers add horizontal lines to visually separate sections of your dashboard. They help create clear boundaries between groups of widgets.

<ExpandAll />

## Default Appearance

A divider with default settings shows a thin horizontal line spanning the full width.

## Tips

- Use between major sections (e.g., between KPI row and chart row)
- Keep dividers minimal — too many dividers make a dashboard feel cluttered
- Combine with [Headers](/layout/header/) for clear section labeling
