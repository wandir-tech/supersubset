# Rows & Columns

Rows group widgets on the same horizontal line. Within a row, each widget's `width` (in grid columns) determines how much horizontal space it takes. Columns can be used for vertical stacking within a row.

## Configuration Options

### Widget Width

Each widget within a row specifies its width in grid columns (1–12). Widgets that exceed the row's column count wrap to the next line.

<ScreenshotComparison
  label="Widget width configuration"
/>

### Widget Height

Each widget specifies its height in pixels. Charts typically use 300–400px; KPI cards use 100–150px.

<ScreenshotComparison
  label="Widget height configuration"
/>

## Tips

- Use **4+4+4** for three equal-width KPI cards
- Use **8+4** for a main chart with a sidebar widget
- Use **12** for full-width tables or headers
- Rows automatically handle responsive wrapping on smaller screens
