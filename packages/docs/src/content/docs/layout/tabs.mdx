# Tabs

Tabs let you organize content into switchable sections within a single dashboard page. Users click tab headers to switch between different views without navigating to a separate page.

## Tips

- Use tabs to **group related content** without creating separate pages
- Keep tab labels short (1–3 words)
- For major content sections, consider [Multi-Page Dashboards](/pages/multi-page/) instead
- Tabs work well for: different time periods, data segments, or metric categories
