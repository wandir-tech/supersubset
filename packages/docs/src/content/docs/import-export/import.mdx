# Import

Load a previously exported dashboard definition from a JSON file. This restores the complete dashboard including all pages, widgets, filters, and interactions.

<ExpandAll />

## Importing a Dashboard

Open the **Import/Export panel** from the designer toolbar and click "Import". Select a JSON file or paste the JSON content.

  label="Import panel and file selection"
  designerSrc={designer_designer}

/>

## After Import

The designer loads the imported dashboard with all its pages and widgets. You can immediately edit or preview it.

  src={designer_default_designer}

## Tips

- Imported dashboards **replace the current dashboard** — export first if you want to keep your work
- The import validates the JSON against the canonical schema — invalid files will show an error
- Use import to **restore from backup**, share between team members, or migrate between environments
