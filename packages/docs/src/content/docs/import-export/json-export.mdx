# JSON Export

Supersubset dashboards are defined by a canonical JSON schema. You can export any dashboard as a JSON file to share with others, back up your work, or version-control your dashboards.

<ExpandAll />

## Exporting a Dashboard

Open the **Import/Export panel** from the designer toolbar and click "Export JSON".

  label="Export panel and JSON output"
  designerSrc={import_export_designer}
  designerAlt="Export panel and JSON output in designer mode"
/>

## The Dashboard Schema

The exported JSON contains the complete dashboard definition including:

- Pages and their layouts
- Widget configurations and field bindings
- Filters and their scopes
- Interactions and navigation rules
- Theme settings

## Tips

- Use export/import for **sharing dashboards** between environments
- The JSON schema is the **canonical source of truth** — it fully describes the dashboard
- Store exported JSON in version control for change tracking
- See [Code View](/import-export/code-view/) to inspect the schema without downloading
